# Logic_Analyzer_STM32F401xC

This project is a simple 4-Channel Logic anaylzer using the STM32F401CCU6 
microcontroller.

Main Objective:
 1. -> Sample signal using GPIO on 4 pins for each timer interrupt
 2. -> Store data in arrays
 3. -> Send data to TTL module using USART
 4. -> Data is processed using a python script
 5. -> Signal diagram displayed using Matplotlib
